# Nim Plugins

Claude Code marketplace for Nim workflows.

## Install

```sh
claude plugin marketplace add jarto666/nim-plugins
claude plugin install nim@nim-plugins
```

## Contents

- `plugins/nim` - Claude plugin for Nim MCP workflows.
- `plugins/nim-architects` - architecture and real-estate workflows that depend on the base Nim plugin.
- `plugins/nim/skills/generate-image` - image generation guidance.
- `plugins/nim/skills/generate-video` - video generation guidance.
- `plugins/nim-architects/skills/client-project-pitch` - imported project pitch workflow skill.
